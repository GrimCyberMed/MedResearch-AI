## Systematic Review Workflow at [University Name]

### Stage 1: Planning (Weeks 1-4)

**Actions**:
1. Develop research question with supervisor/collaborators
2. Conduct scoping search to assess feasibility
3. Register with [Research Office]
4. Apply for ethics approval (if needed)
5. Apply for internal funding (if needed)

**Deliverables**:
- Research question (PICO/SPIDER)
- Scoping search results
- Ethics application (if applicable)
- Funding application (if applicable)

**University Resources**:
- [Resource 1]
- [Resource 2]

### Stage 2: Protocol Development (Weeks 5-8)

**Actions**:
1. Develop full protocol using [template]
2. Consult with [research librarian/support service]
3. Register protocol on PROSPERO
4. Submit protocol for approval

**Deliverables**:
- Complete protocol document
- PROSPERO registration number
- Approved search strategies
- Data extraction form

**University Resources**:
- [Resource 1]
- [Resource 2]

### Stage 3: Searching (Weeks 9-12)

**Actions**:
1. Execute searches in all planned databases
2. Export citations to reference manager
3. Deduplicate citations
4. Document search process

**Deliverables**:
- Citation library (deduplicated)
- Search documentation
- PRISMA flow diagram (identification stage)

**University Resources**:
- [Database access]
- [Reference manager]
- [Interlibrary loan]

### Stage 4: Screening and Selection (Weeks 13-20)

**Actions**:
1. Title/abstract screening (dual independent)
2. Full-text review (dual independent)
3. Resolve conflicts
4. Document exclusions

**Deliverables**:
- Included studies list
- Excluded studies log with reasons
- PRISMA flow diagram (complete)

**University Resources**:
- [Screening tool]
- [Research assistant support]

### Stage 5: Data Extraction (Weeks 21-32)

**Actions**:
1. Pilot extraction form
2. Dual independent extraction
3. Risk of bias assessment
4. Contact authors for missing data

**Deliverables**:
- Extraction database
- Risk of bias assessments
- Author contact log

**University Resources**:
- [Data extraction tool]
- [Statistical consultation]

### Stage 6: Analysis and Synthesis (Weeks 33-40)

**Actions**:
1. Descriptive synthesis
2. Meta-analysis (if appropriate)
3. Subgroup and sensitivity analyses
4. GRADE assessment

**Deliverables**:
- Analysis results
- Forest plots
- GRADE evidence profile

**University Resources**:
- [Statistical software]
- [Statistical consultation]
- [Training materials]

### Stage 7: Writing and Dissemination (Weeks 41-52)

**Actions**:
1. Write manuscript (PRISMA-compliant)
2. Internal peer review
3. Submit to journal
4. Deposit in institutional repository

**Deliverables**:
- Complete manuscript
- Supplementary materials
- PRISMA checklist

**University Resources**:
- [Editing service]
- [Publication support]
- [Open access funding]

## Customization Instructions

**To customize this template for your institution**:

1. **Replace all [bracketed placeholders]** with institution-specific information
2. **Update contact information** with current details
3. **Verify database subscriptions** with library
4. **Confirm funding programs** with research office
5. **Check ethics requirements** with ethics committee
6. **Update timeline** based on institutional norms
7. **Add institution-specific policies** as needed
8. **Remove sections** not applicable to your institution
9. **Add sections** for unique institutional requirements
10. **Review and approve** with relevant departments

**Key Sections to Customize**:
- Institution Overview (name, location, focus areas)
- Database Access (verify current subscriptions)
- Funding Programs (amounts, deadlines, eligibility)
- Contact Information (all emails, phones, names)
- Templates and Resources (links to institutional resources)
- Compliance Checklist (institution-specific requirements)

**Maintenance**:
- Review annually or when policies change
- Update contact information as staff changes
- Verify database subscriptions each academic year
- Update funding programs as they change
- Revise timeline based on actual completion times

**Questions or Assistance**:
Contact [institutional research support office] for help customizing this template.
